Fade In and Fade Out

The fadeIn() method is used to fade in hidden elements. fadeOut() method is used to fade out elements that are displayed. This is demonstrated in the example here. PS: remember to include a jQuery CDN if library is not downloaded.

Fade Toggle Method

fadeToggle() method toggles between fadeIn() and fadeOut() methods. If the elements are faded out, fadeToggle() will fade them in. Else if the elements are fade in, the fadeToggle() method will fade them out.
